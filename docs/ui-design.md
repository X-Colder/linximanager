# 灵犀掌柜 - UI/UX设计规范文档

## 1. 设计系统 (Design System)

### 1.1 颜色规范

#### 主色
| 用途 | 色值 | 说明 |
|:---|:---|:---|
| 品牌主色 | `#4A6CF7` | 信任感、科技感，用于主按钮、导航高亮 |
| 主色浅 | `#E8EDFF` | 主色背景、选中态背景 |
| 主色深 | `#3451D1` | hover态、重要标题 |

#### 语义色
| 用途 | 色值 | 说明 |
|:---|:---|:---|
| 成功/安全 | `#10B981` | 审核通过、库存充足、支付成功 |
| 警告 | `#F59E0B` | 积压预警(黄色)、待处理 |
| 危险/紧急 | `#EF4444` | 缺货预警(红色)、删除、错误 |
| 信息 | `#3B82F6` | 临期提醒(蓝色)、引导信息 |
| 促销 | `#F97316` | 促销标签、折扣价格 |

#### 预警三色
| 预警级别 | 色值 | 背景色 | 场景 |
|:---|:---|:---|:---|
| 红色预警 | `#EF4444` | `#FEF2F2` | 24/48h内缺货 |
| 黄色预警 | `#F59E0B` | `#FFFBEB` | 库存积压 |
| 蓝色预警 | `#3B82F6` | `#EFF6FF` | 1-3天内过期 |

#### 中性色
| 用途 | 色值 |
|:---|:---|
| 标题文字 | `#1F2937` |
| 正文文字 | `#4B5563` |
| 辅助文字 | `#9CA3AF` |
| 边框 | `#E5E7EB` |
| 分割线 | `#F3F4F6` |
| 页面背景 | `#F9FAFB` |
| 卡片背景 | `#FFFFFF` |

### 1.2 字体规范

#### Web管理后台
| 层级 | 字号 | 字重 | 行高 | 用途 |
|:---|:---|:---|:---|:---|
| H1 | 24px | 700 | 32px | 页面标题 |
| H2 | 20px | 600 | 28px | 模块标题 |
| H3 | 16px | 600 | 24px | 卡片标题 |
| Body | 14px | 400 | 22px | 正文内容 |
| Caption | 12px | 400 | 18px | 辅助说明 |
| Data | 28px | 700 | 36px | 数据指标 |

#### 小程序端
| 层级 | 字号 | 字重 | 用途 |
|:---|:---|:---|:---|
| H1 | 36rpx | 700 | 页面标题 |
| H2 | 32rpx | 600 | 模块标题 |
| Body | 28rpx | 400 | 正文 |
| Caption | 24rpx | 400 | 辅助 |
| Data | 48rpx | 700 | 核心数据 |

字体族：`-apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', sans-serif`

### 1.3 间距系统

基准网格：**8px**

| Token | 值 | 用途 |
|:---|:---|:---|
| xs | 4px | 图标与文字间距 |
| sm | 8px | 元素内间距 |
| md | 16px | 卡片内边距、列表项间距 |
| lg | 24px | 模块间距 |
| xl | 32px | 页面边距(Web) |
| 2xl | 48px | 大模块分隔 |

小程序页面边距：`32rpx`

### 1.4 圆角与阴影

| 元素 | 圆角 | 阴影 |
|:---|:---|:---|
| 按钮 | 8px | 无 |
| 卡片 | 12px | `0 2px 8px rgba(0,0,0,0.06)` |
| 弹窗 | 16px | `0 8px 32px rgba(0,0,0,0.12)` |
| 头像 | 50% | 无 |
| 输入框 | 8px | 内阴影 `inset 0 1px 2px rgba(0,0,0,0.05)` |
| 标签Tag | 4px | 无 |

### 1.5 图标风格

- 风格：线性图标(Outline)，2px线宽
- 尺寸：16px / 20px / 24px / 32px
- 推荐图标库：Remix Icon / Phosphor Icons
- 预警图标带颜色填充

## 4. 交互规范

### 4.1 按钮状态

| 状态 | 表现 |
|:---|:---|
| 正常 | 品牌色背景、白色文字、圆角8px |
| Hover | 背景色加深10%、微微上移1px |
| 按下 | 背景色加深20%、缩放0.98 |
| 禁用 | 灰色背景(`#D1D5DB`)、不可点击 |
| 加载中 | 文字替换为spinner + "处理中..." |

按钮尺寸：
- 大：高度48px(小程序88rpx)，全宽
- 中：高度40px，自适应宽度
- 小：高度32px，紧凑场景

### 4.2 表单交互

- 输入框高度：40px(Web) / 80rpx(小程序)
- 必填标记：标签前红色星号 `*`
- 实时校验：失效时输入框变红 + 下方红色提示文字
- 语音输入：长按麦克风按钮，松开识别，识别结果可编辑

### 4.3 弹窗/抽屉规范

| 类型 | 使用场景 | 规范 |
|:---|:---|:---|
| Modal | 确认操作（删除、冻结）| 居中、背景遮罩40%黑 |
| Drawer | 详情/编辑面板 | 右侧滑出、宽度50%(Web) |
| ActionSheet | 小程序底部选择 | 底部弹出、圆角16px |
| Toast | 轻量反馈 | 顶部浮出、2秒消失 |

### 4.4 加载状态

| 场景 | 方案 |
|:---|:---|
| 页面首次加载 | 骨架屏(Skeleton)，模拟内容布局 |
| 列表加载更多 | 底部spinner + "加载中" |
| 按钮提交 | 按钮内spinner替换文字 |
| AI计算中 | 专属动画 + "AI思考中..."(带预计时间) |
| 下拉刷新 | 自定义动画(灵犀Logo旋转) |

### 4.5 空状态

| 场景 | 插图 | 文案 | 操作 |
|:---|:---|:---|:---|
| 无商品 | 空货架 | "还没有商品，开始添加吧" | [添加商品] |
| 无订单 | 空箱子 | "暂无订单" | [去逛逛] |
| 无预警 | 绿色对勾 | "一切正常，继续保持！" | 无 |
| 网络错误 | 断线插画 | "网络不给力" | [重试] |
| 服务器错误 | 维修工人 | "系统维护中，请稍后" | [重试] |

### 4.6 Toast/Notification

- **Toast**：轻提示，2秒自动消失，不可交互
  - 成功：绿色图标 + 文字
  - 失败：红色图标 + 文字
- **Notification**：重要通知，可关闭
  - 顶部滑入，带操作按钮
  - 5秒后自动收起(不消失)

```

  - subtitle: string       // "剩余2kg"
  - description: string    // "预计明天缺货"

样式:

  - 圆角12px
  - 内边距16px
  - 右侧箭头引导操作
```

### 5.3 库存沙盘组件 (InventoryShelf)

```
Props:
  - items: InventoryItem[]
  - viewMode: 'shelf' | 'list'
  - onItemClick: (item) => void

InventoryItem:
  - name: string
  - currentStock: number
  - maxStock: number       // 满水位线
  - safetyStock: number
  - unit: string
  - alertLevel?: 'red' | 'yellow' | 'blue'

交互:
  - 每个格子显示填充比例(渐变色)
  - 低于安全线显示红色
  - 点击进入商品详情
  - 支持拖动调整满水位线
  - 长按快捷操作(入库/出库)
```

```
Props:

PromoPlan:
  - name: string           // "方案B"
  - label: string          // "推荐"
  - discount: number       // 0.7
  - estimatedDays: number
  - estimatedProfit: number
  - reason: string

样式:
  - 推荐方案带金色边框 + ⭐标记
  - 亏损方案利润数字红色
  - 选中方案高亮(主色边框)
  - 底部执行按钮(推荐方案为主色填充，其他为描边)
```

```
样式:
  - 横向布局: 左图(80x80) + 右侧信息
  - 促销价橙色、原价灰色划线
  - 标签圆角pill样式
  - 右下角 [+] 圆形按钮(加入购物车)
```

## 7. 动效规范

|:---|:---|:---|

## 8. 无障碍设计

- 色彩对比度 ≥ 4.5:1 (WCAG AA)
- 预警不仅用颜色区分，同时用图标和文字描述
- 触摸目标最小 44x44px
- 语音操作作为键盘输入的替代方案
- 图片提供alt文本
